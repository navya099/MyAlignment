# myalignment Python Package

myalignment는 파이썬 기반의 선형 패키지로, 원곡선 ,직선, 완화곡선 등 기본적인 세그먼트 객체를 다룰 수 있도록 설계되었습니다.  
다양한 프로젝트에서 공용 라이브러리로 활용할 수 있습니다.

## Features
- `alignment`,: 고수준 선형 객체
- `segment`,: 선형을 이루는 단위 객체

## Dependence
autocad: point2d
